# mxbi dotfiles

> ⚠️ **WIP** - Work in Progress

Cross-platform configuration and bootstrap scripts for production environments.

## Features

- **Bash-first**: Optimized for bash in production environments (macOS/Linux)
- **PowerShell for Windows**: Uses pwsh on Windows platforms
- **Cross-platform**: Supports macOS, Linux, and Windows
- **Interactive installation**: Choose what to install

## Quick Start

```bash
git clone https://github.com/yourusername/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
# macOS / Linux
./scripts/install.sh
# Windows (pwsh)
.\scripts\install.ps1
```

## Platform Support

- ✅ macOS (bash)
- ✅ Linux (bash)
- ✅ Windows (PowerShell/pwsh)
