# mxbi dotfiles

> ⚠️ **WIP** - 开发中

跨平台配置与启动脚本，用于生产环境。

## 特性

- **Bash 优先**: 针对 macOS/Linux 生产环境 bash 优化
- **PowerShell for Windows**: Windows 平台使用 pwsh
- **跨平台**: 支持 macOS、Linux、Windows
- **交互式安装**: 选择需要安装的组件

## 快速开始

```bash
git clone https://github.com/yourusername/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
# macOS / Linux
./scripts/install.sh
# Windows (pwsh)
.\scripts\install.ps1
```

## 平台支持

- ✅ macOS (bash)
- ✅ Linux (bash)
- ✅ Windows (PowerShell/pwsh)
